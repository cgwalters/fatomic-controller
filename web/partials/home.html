The Server Atomic Controller Fedora Remix is a minimal operating system, designed to host Docker containers. Using technology from the Fedora Atomic Initiative, it supports atomic upgrades.

You can run pre-built images on VirtualBox, QEMU/KVM, and OpenStack. Or set up your own build server, and include your own custom RPM content such as a monitoring agent, or your favorite editor.

SELinux in enforcing mode

Because the underlying rpm-ostree technology is an evolution of Fedora and RPM, Controller takes advantage of all of the work that's been put into things like Security-Enhanced Linux. This allows it to confine Docker containers with strong mandatory access control.

Smaller than RPM can normally go.

With RPM, removing things like extra documentation and translations has always been suboptimal. OSTree allows Controller to use RPMs to build, but then strip out these things to get a more minimal host operating system - and updates only get more efficient as the system becomes smaller.

Easily create your own customized controllers

You are not locked into our vision of what packages should go into a core controller operating system. Use the rpm-ostree technology to create your own! You can easily take our existing images, and add your own repository, configuring your servers to pull from it.
